# 🌸 PetalOS

PetalOS is a WebOS. basically an operating system but web-based.

PetalOS is maybe a lil bit based on/inspired by Windows, macOS, and Kali Linux. I'm sorry if there's no bold resemblance of any of these insipiration OS but yeah, love these OS!

Built exclusively for Hack Club Stardance.

---

## Features (Planned?)

* Desktop environment (ofc)
* Window management system?
* Dock and app launcher
* Some simple applications
* Virtual file system
